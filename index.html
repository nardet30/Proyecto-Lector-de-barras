Proyecto Lector de Barras

Creado por Bernardo Moscardo (@nardo30)

Descripción

Este sistema interactivo permite la lectura de códigos de barras de una dimensión (1D) utilizando la cámara web de cualquier ordenador. Implementado en Python con OpenCV, ofrece una experiencia de usuario fluida y profesional.

Guía de Instalación

  • Descarar o Clonar: Descarga el código desde GitHub o clona el repositorio: git clone https://github.com/nardo30/Proyecto-Lector-de-barras.git
  • Preparar el entorno: Instala las dependencias necesarias con pip: pip install opencv-python numpy
  • Lanzar la aplicación: Ejecuta el script de Python: python lector_barras.py

Cómo Funciona

Al iniciar, verás un panel interactivo. Coloca tu producto frente a la cámara y haz clic en el botón naranja "ESCANEAR". La aplicación procesará la imagen, detectará el código y emitirá un pitido de confirmación.
